About the project

Indra is an agent-based modeling (ABM) system in Python.


The ultimate goal is to simplify ABM creation so that a model can be specified in only a few lines of code, or by filling in a few parameters on a screen. To that end we plan to build a Web front-end that will allow non-coders to build models, while at the same time permitting coders to use Python for more flexibility in model creation.
